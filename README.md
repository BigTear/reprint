# reprint

---

## 终端输出替换工具

支持 Win/Linux/macOS

---

🏠 [主页](https://github.com/BigTear/reprint) 📚 [文档](https://github.com/BigTear/reprint)

---

### 用途

- 给不支持输出中文的软件添加翻译并无缝使用（我开发这个程序的目的）
- 给终端添加无障碍支持

---

### 源码说明

| 项目             | 说明      | 备注  |
|----------------|---------|-----|
| `r`            | 主程序     | bin |
| `r-data`       | 配置数据存储  | bin |
| `r-editor`     | 配置编辑器   | bin |
| `r-library`    | 共享、下载配置 | bin |
| `r-translator` | 翻译器     | bin |
| `common`       | 共享库     | lib |
